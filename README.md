SuperSnake
============

![alt tag](https://github.com/DidelotK/SuperSnake/blob/master/README/readme.png)

To deploy the vm:
-----------------

- cd vagrant
- vagrant up --provision

Role installed:

- Nginx
- Sqlite
- Supervisord
- gunicorn
- flask

Required
--------

- vagrant (With plugin vagrant-vbguest)
- ansible
